# dim
## Definition
small package for display information about images in selected folder.
output format:
```
image.png
<area x="-212.5" y="-27" w="425" h="54">
```
## Dependencies
[image-size](https://www.npmjs.com/package/image-size) - module to get dimensions of any image file.
## Command-Line Usage
```
$ dim
```
